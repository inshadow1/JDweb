### enum

```ts
// 利用位运算的特性，二进制中，每位是独立的，
enum FutureFlags{
  None   = 0,
  Happy  = 1,
  Rich   = 1<<1,
  Wisdom = 1<<2,
  Famous = 1<<3,
  GreatHuman = Happy|Rich|Wisdom|Famous
}
interface Human{
  flags:FutureFlags
  [key:string]:any
}
animal.flags |= AnimalFlags.HasClaws | AnimalFlags.CanFly;
```

- 我们使用 `|=` 来添加一个标志
- 组合使用 `&=` 和 `~` 来清理一个标志
- `|` 来合并标志。
