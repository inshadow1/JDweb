小程序架构

>配置文件：.json文件：对于固定功能进行设置头部文件，头部的颜色，字体，button的设置
>
>逻辑：.js文件，开发当中用到的属性，方法，配置信息
>
>结构：.wxml文件，布局flex布局
>
>样式：.wxss文件，类似css，并提供了rpx
>
>模块：写一些通用的功能，CommonJS规范
>
>WXS：小程序当中提供的固定的自定义JS功能，在结构中使用

> 组件：
>
> - 内置组件
> - 自定义组件
> - 第三方组件
>
> 插件：
>
> - 自定义插件
> - 第三方插件
>
> 云开发（比较重要的概念，弱化服务端概念）
>
> - 云数据库
> - 云存储
> - 云函数
> - HTTP API
> - 云调用
>
> API
>
> - 小程序端
> - 服务端
