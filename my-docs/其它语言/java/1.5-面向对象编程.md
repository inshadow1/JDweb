## 数组

数组是将一系列数据并排放置，依次向后排

声明一个数组

```java
public class StudyFall {
  public static void main(String[] args){
    // 表明是 double 类型组成的一个数组
    double[] height = {10,20,31,44};
    // 也可以通过 new 创建一个数组
    double[] newHeight = new double[5]
  }
}
```
